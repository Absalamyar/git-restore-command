for the restoreing the file before staging area we are using this
git restore file name or dott which is for all file
ex:
1: git restore index.html
2" git restore .
 and for restoring file or code from staging area it means when enter git add . code we use this
git restore --staged file name or dott which is used for all file

2: git restore --staged .
